📚 dir2opds

Self-hosted OPDS ebook server — turn any folder into a digital library. No database, no Calibre. Just your books, served.

OPDS 1.1 Compliant
Get Started View on GitHub Download

⚡ Quick Start

Run your own ebook server in seconds with Docker:

docker run -d -p 8080:8080 -v ./books:/books --name dir2opds ghcr.io/dubyte/dir2opds:latest

Or install with Go:

go install github.com/dubyte/dir2opds@latest
dir2opds -dir /path/to/books -port 8080

✨ Why dir2opds?

🗂️ No Database

Reads directly from your filesystem. No Calibre, no complex setup. Just point it at a folder.

📱 Any Client

Works with Moon+ Reader, Cantook, KYBook, and any OPDS-compatible ebook reader app.

🏠 Self-Hosted

Run on a Raspberry Pi, NAS, home server, or VPS. Keep your library private and local.

🐳 Container Ready

Single binary or Docker image. Lightweight and ideal for headless servers.

🔍 Search & Covers

Optional filename search, EPUB/PDF metadata extraction, and cover image support.

⚡ Fast & Cached

ETag/Last-Modified headers, gzip compression, and pagination for large catalogs.

📖 What is OPDS?

OPDS (Open Publication Distribution System) is an open standard for cataloging and distributing ebooks. OPDS clients can browse, search, and download books from any OPDS server — including dir2opds, your personal self-hosted ebook library.

🛠️ Supported Formats

  • 📕 EPUB — with optional metadata and cover extraction
  • 📗 PDF — with optional title/author extraction
  • 📘 MOBI / AZW3 — via configurable MIME types
  • 📙 CBZ / CBR — comic book archive support
  • 📄 FB2, TXT, and more — customizable via MIME map
